```
> remax --help

Usage: remax-cli <command> [options]

命令：
  remax-cli build  build your project

选项：
  --help        显示帮助信息                                               [布尔]
  --version     显示版本号                                                 [布尔]
  --watch, -w   监听文件变化                                [布尔] [默认值: false]
  --target, -t  目标平台, 如 wechat, alipay                        [字符串] [必需]
```
